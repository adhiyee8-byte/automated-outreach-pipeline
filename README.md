# 🚀 Automated Outreach Pipeline

An AI-powered outreach automation system that streamlines B2B lead generation and personalized email outreach.

## 📌 Problem Statement

Businesses spend significant time identifying similar companies, finding key decision makers, and drafting personalized outreach emails. This manual process is repetitive and time-consuming.

This project automates the outreach workflow using AI-driven company discovery and email generation.

---

## ✨ Features

- Input a company domain
- Discover similar companies
- Identify decision makers
- Generate personalized outreach emails
- Simple web interface using Streamlit

---

## 🔄 Workflow

Company Domain → Similar Companies → Decision Makers → Outreach Email Generation

---

## 🛠 Tech Stack

- Python
- Streamlit
- Google Colab
- ngrok
- AI/LLM Integration

---

## 🚀 How to Run

### 1. Clone Repository

```bash
git clone <your-repo-link>
cd automated-outreach-pipeline
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run Application

```bash
streamlit run app.py
```
